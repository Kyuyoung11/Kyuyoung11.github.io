## Claude Code란?

**Claude Code**는 Anthropic이 만든 AI 코딩 어시스턴트 CLI 도구입니다.  
터미널에서 직접 실행하며, 코드 작성 / 버그 수정 / 리팩토링 / 파일 탐색 등 개발 관련 작업을 자연어로 지시할 수 있습니다.

단순히 코드 한 줄을 생성해주는 수준이 아니라, **프로젝트 전체 구조를 파악하고 여러 파일을 동시에 수정**할 수 있다는 게 핵심입니다.

<br>

### 기존 AI 코딩 도구와의 차이점

| | ChatGPT / 일반 AI | GitHub Copilot | Claude Code |
|---|---|---|---|
| 사용 방식 | 웹 채팅 | IDE 플러그인 | 터미널 CLI |
| 파일 접근 | 직접 불가 | 현재 파일만 | 프로젝트 전체 |
| 코드 수정 | 복붙 필요 | 인라인 제안 | 직접 수정 |
| 명령 실행 | 불가 | 불가 | 가능 (git, npm 등) |

## 설치 방법

### 사전 조건
- Node.js 18 이상

<br>

### 설치

````bash
npm install -g @anthropic-ai/claude-code
````

<br>

### 실행

````bash
claude
````

처음 실행하면 Anthropic 계정 로그인 과정이 진행됩니다.  
브라우저가 자동으로 열리고, 로그인 후 CLI로 돌아오면 바로 사용할 수 있습니다.

<br><br>

## 기본 사용법

### 프로젝트 폴더에서 시작하기

원하는 프로젝트 폴더로 이동한 다음 실행하는 것이 기본입니다.

````bash
cd my-project
claude
````

Claude Code는 현재 폴더를 기준으로 프로젝트를 파악합니다.

<br>

### 자연어로 작업 지시하기

대화창이 열리면 자연어로 지시하면 됩니다.

````
> 이 프로젝트의 구조를 설명해줘
> UserController에서 이메일 중복 체크 로직이 빠져있어. 추가해줘
> git status 확인하고 변경사항 커밋해줘
````

<br>

### 주요 슬래시 명령어

| 명령어 | 설명 |
|---|---|
| `/help` | 사용 가능한 명령어 목록 |
| `/clear` | 대화 초기화 |
| `/compact` | 대화 내용 요약 (토큰 절약) |
| `/exit` | 종료 |

## 실제로 써보니

### 좋았던 점

**파일을 직접 수정해줌**  
"A 파일에서 B 함수를 수정해줘" 하면 실제로 파일을 열어서 고쳐줍니다.  
복붙할 필요 없이 바로 반영됩니다.

<br>

**프로젝트 구조를 스스로 파악함**  
처음에 아무것도 설명하지 않아도, 파일들을 훑어보고 어떤 프로젝트인지 파악합니다.  
"이 프로젝트에서 인증 관련 코드가 어디에 있어?" 같은 질문도 잘 찾아줍니다.

<br>

**git 작업도 같이 해줌**  
커밋 메시지 작성, 스테이징, 커밋까지 한 번에 처리해줄 수 있습니다.

<br>

### 아직 익숙하지 않은 점

**승인 요청이 자주 뜸**  
파일 수정이나 명령어 실행 전에 "이 작업을 해도 될까요?" 하고 확인을 요청합니다.  
처음엔 번거롭게 느껴지지만, 의도치 않은 수정을 막아주는 안전장치이기도 합니다.

<br>

**대화가 길어지면 느려질 수 있음**  
대화가 쌓일수록 컨텍스트가 늘어나 응답이 조금 느려지는 경우가 있습니다.  
이때는 `/compact` 명령으로 대화를 요약하면 도움이 됩니다.

## 정리

Claude Code는 단순한 코드 생성 도구가 아니라, **터미널에서 함께 개발하는 AI 동료**에 가깝습니다.  
처음에는 어색하지만, 한번 흐름에 익숙해지면 반복적인 작업 시간을 크게 줄여줍니다.
